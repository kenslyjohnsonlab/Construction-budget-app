# Construction Budget Tracker

A simple web app for tracking construction project budgets, costs, and projected profit.

## 🚀 Overview

This app helps builders, developers, and investors:

- Track project budgets
- Monitor actual costs
- Calculate remaining budget
- Estimate profit in real time

Built as a simple MVP to replace spreadsheets and reduce costly mistakes.

---

## 🛠 Tech Stack

- Next.js (Frontend + Backend)
- React
- Tailwind CSS
- GitHub (Code storage)
- Railway (Deployment)

---

## 📱 Features (MVP)

- Create and edit project info
- Input:
  - Land cost
  - Hard cost budget
  - Soft costs
  - Draws received
- Add and remove cost items
- Automatic calculations:
  - Total budget
  - Actual costs
  - Remaining budget
  - Projected profit

---

## 🧠 How It Works

1. Enter your project details
2. Add your budget numbers
3. Track costs as they come in
4. View your real-time profit and budget status
